# eMailer
Android Library to send eMails

Add it in your root build.gradle at the end of repositories:


allprojects {
		
		repositories {
		
			...
			
			maven { url 'https://jitpack.io' }
			
		}
		
	}

  
  Add the dependency:
  
  
  dependencies {
  
	        implementation 'com.github.cyberman-io:eMailer:v1.0'
		
	}
